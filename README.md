# 📚 BookStore

A modern bookstore web application built with HTML, CSS, and JavaScript. The project demonstrates responsive web design, interactive user interfaces, and dynamic content rendering using vanilla JavaScript.

---

## 🚀 Features

- Browse a collection of books
- Responsive design for desktop and mobile devices
- Interactive shopping cart
- Dynamic book rendering
- Price calculation
- Clean and user-friendly interface
- Vanilla JavaScript implementation without external frameworks

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/YourUsername/bookstore.git
```

Open the project folder:

```bash
cd bookstore
```

Start the application by opening the `index.html` file in your browser.
